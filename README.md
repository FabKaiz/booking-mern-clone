# Booking MERN clone

### Booking MERN clone with admin panel using
* MongoDB
* Express
* React
* Node
* React Router V6* 
* axios
* Cloudinary
* bcryptJs
* cookie-parser
* jsonwebtoken
* Sass
* Material Ui Icons - Table - Data-grid
* Recharts
